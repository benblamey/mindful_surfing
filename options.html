Mindful Surfing

A Chrome browser extension to help avoid wasting excessive time - or retain "mindfulness" when browsing. Unlike other extensions which block time on specific 'distracting' sites, this extension instead focuses on the time the browser has been running.

Show a notification once when the browser has been running for minutes.

And every minutes thereafter.

Shutdown the browser after minutes.

Warning - after initial warning notifications, there is no confirmation for the browser close!

If you decide to enable this feature, you might wish to enable persistence of the tabs between browser sessions, see: Settings > On startup > "continue where you left off"
